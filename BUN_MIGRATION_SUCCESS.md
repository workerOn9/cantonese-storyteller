## 🎯 使用指南

### 快速开始
```bash
# 1. 设置环境
export DATABASE_URL="sqlite:$(pwd)/src-tauri/app.db"

# 2. 启动开发环境
bun run tauri:dev

# 3. 在浏览器中访问: http://localhost:1420/
```

### 完整工作流
